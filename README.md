# 一、概述
* uniapp项目
* 兼容android,ios、微信小程序、h5

# 二、项目结构
项目基本目录结构及各目录用途如下：

|目录|功能|
|--	|--	|
|common|公用目录，主要包括助手函数，公用样式等|
|components|组件存放目录|
|data|页面数据目录，存放所有页面数据|
|pages|用于存放所有页面|
|static/mock|用于模拟后端获取的静态资源|
|uni_modules|用于存放uni_modules组件|
